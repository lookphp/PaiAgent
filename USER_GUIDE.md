## 产品简介

PaiAgent 是一个可视化的 AI Agent 工作流平台，让您可以：

- 🎨 **拖拽式编辑**: 通过拖拽节点创建 AI 工作流
- 🤖 **大模型集成**: 支持通义千问、DeepSeek 等多种大模型
- 🎙️ **音频合成**: 将 AI 响应转换为超拟人音频
- 🐛 **实时调试**: 查看执行日志，快速定位问题
- 🔊 **音频播放**: 直接播放合成的 AI 播客内容

### 典型使用场景

```
用户输入 → 大模型处理 → 音频合成 → 播放 AI 播客
```

## 快速入门

### 第一步：启动服务

**启动后台服务:**
```bash
cd pai-agent-server
mvn spring-boot:run
```

**启动前端服务:**
```bash
cd pai-agent-frontend
npm run dev
```

### 第二步：访问应用

打开浏览器访问：`http://localhost:5173`

### 第三步：创建第一个工作流

1. 从左侧节点库拖拽节点到画布
2. 连接节点（从源节点的底部拖到目标节点的顶部）
3. 点击节点配置参数
4. 点击右上角"调试"按钮
5. 输入测试文本，点击"运行工作流"
6. 等待执行完成，播放音频

## 操作指南

### 1. 添加节点

**方法一：拖拽添加**
1. 从左侧节点库点击并按住节点
2. 拖拽到画布中央
3. 松开鼠标，节点即添加到画布

**方法二：双击添加**（如支持）
1. 双击左侧节点库中的节点
2. 节点自动添加到画布中央

### 2. 连接节点

1. 找到源节点底部的连接点（圆点）
2. 点击并拖拽连线到目标节点顶部的连接点
3. 松开鼠标，连接建立

**连接规则:**
- 输入节点只能作为起点（只有输出连接点）
- 输出节点只能作为终点（只有输入连接点）
- 大模型节点和工具节点可以有输入和输出

### 3. 删除节点

**删除单个节点:**
1. 点击选中节点
2. 按键盘 `Delete` 或 `Backspace` 键

**删除多个节点:**
1. 框选多个节点
2. 按键盘 `Delete` 键

**删除连线:**
1. 点击选中连线
2. 按键盘 `Delete` 键

### 4. 配置节点

1. 点击画布中的节点
2. 右侧自动打开配置抽屉
3. 修改配置参数
4. 点击"保存配置"

### 5. 配置画布

**平移画布:**
- 按住鼠标右键拖拽，或使用方向键

**缩放画布:**
- 滚动鼠标滚轮，或使用 `+` `-` 键

**适应画布:**
- 点击右下角适应画布按钮

### 6. 运行工作流

1. 点击右上角"运行"按钮
2. 右侧调试抽屉自动打开
3. 在输入框中输入测试文本
4. 点击"运行工作流"按钮
5. 查看执行日志
6. 执行完成后播放音频

## 工作流示例

### 示例一：AI 播客生成

**目标**: 将用户输入转换为 AI 播报的音频

**节点配置:**

```
┌─────────┐    ┌───────────┐    ┌──────────────┐    ┌─────────┐
│  用户   │───►│  通义千问  │───►│ 超拟人音频   │───►│  输出   │
│  输入   │    │  大模型    │    │ 合成工具     │    │  节点   │
└─────────┘    └───────────┘    └──────────────┘    └─────────┘
```

**配置步骤:**

1. **输入节点** (自动添加)
   - 无需配置

2. **通义千问节点**
   - 模型选择：`通义千问-Max`
   - 系统提示词：`你是一个专业的 AI 助手，请用简洁清晰的语言回答用户问题。`

3. **超拟人音频合成节点**
   - 工具类型：`超拟人音频合成`
   - 音色选择：`女声 1 号`

4. **输出节点** (自动添加)
   - 输出格式：`音频`

**测试运行:**
- 输入：`请介绍一下什么是人工智能？`
- 预期输出：AI 语音回答

## 常见问题

### Q1: 节点拖拽后不显示怎么办？

**A:** 请检查以下几点:
1. 确保浏览器支持拖拽功能
2. 刷新页面重试
3. 检查控制台是否有报错信息

### Q2: 连线时无法连接怎么办？

**A:** 请确认:
1. 源节点必须有输出连接点（底部）
2. 目标节点必须有输入连接点（顶部）
3. 不能从节点连接到自身
4. 不能创建环路

### Q3: 运行时提示"请先添加节点"？

**A:** 画布必须至少有一个节点才能运行。请从左侧节点库拖拽节点到画布。

### Q4: 调试面板没有显示日志？

**A:** 请检查:
1. 后台服务是否正常启动
2. 网络连接是否正常
3. 浏览器控制台是否有错误

### Q5: 音频无法播放？

**A:** 请确认:
1. 工作流中包含音频合成节点
2. 浏览器支持音频播放
3. 检查执行日志是否有错误

### Q6: 配置保存后不生效？

**A:** 请尝试:
1. 确保点击了"保存配置"按钮
2. 重新选中节点查看配置
3. 刷新页面重新配置

### Q7: 如何清空画布？

**A:** 点击右上角工具栏的"清空"按钮，确认后即可清空画布。

### Q8: 支持撤销操作吗？

**A:** 当前版本暂不支持撤销功能，请谨慎操作。

## 配置说明

### 前端配置

前端无需额外配置，如需修改 API 地址，请编辑:

**文件**: `pai-agent-frontend/src/services/api.ts`

```typescript
const API_BASE_URL = 'http://localhost:8080/api';
// 修改为实际的后台地址
```

### 后台配置

**文件**: `pai-agent-server/src/main/resources/application.properties`

```properties
# 服务端口
server.port=8080

# AI API 密钥配置
paiagent.ai.qwen.api-key=你的通义千问 API 密钥
paiagent.ai.deepseek.api-key=你的 DeepSeek API 密钥
paiagent.ai.audio-synthesis.api-key=你的音频合成 API 密钥

# 数据库配置 (生产环境请修改为 MySQL)
spring.datasource.url=jdbc:h2:mem:paiagent
spring.h2.console.enabled=true
```

### 环境变量配置（推荐）

生产环境建议使用环境变量：

```bash
# Linux/Mac
export QWEN_API_KEY=your-api-key
export DEEPSEEK_API_KEY=your-api-key
export AUDIO_SYNTHESIS_API_KEY=your-api-key

# Windows
set QWEN_API_KEY=your-api-key
set DEEPSEEK_API_KEY=your-api-key
set AUDIO_SYNTHESIS_API_KEY=your-api-key
```
